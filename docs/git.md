# Creación repositorio de Github

En este apartado se mostrará como crear un repositorio de github y modificaremos su infraestructura. Primero creamos el repositorio desde github como se muestra en la imagen:![creación repositorio](./img/creacion-repo.png)

Una vez creado el repositorio lo vamos a clonar con `git clone` mediante ssh para tenerlo en nuestro equipo personal. Para crear la infraestructura del proyecto hemos clonado un repositorio de una actividad anterior muy parecida a esta
y hemos borrado y añadido algunos archivos según queríamos presentar el repositorio. Una vez realizado los cambios hemos añadido los archivos nuevos con los comandos `git add .` y `git commit`, en la siguiente imagen se muestra mejor:
![infraestructura del repositorio](./img/infraestructura-repo.png)

Luego para añadirlo a el repositorio original en github hemos utilizado el comando `git push origin main`.
